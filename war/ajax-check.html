Warning: This way lies madness.

This is an experimental rewrite of almost all of the Photoprism web site functions. It uses a new programming style called "AJAX," which is the hip way of saying "it uses a hell of a lot of Javascript". The new style pushes most of the programming logic and control functions into your browser, instead of having the server do all the work. The advantage is that if your browser doesn't bungle it up, the site can be much faster and more interactive (because it doesn't wait for the server to generate an entire new page every time you click something). The disadvantage is that if your browser bungles it up, nothing works.

Fancy new sites like Gmail and, sigh, flickr depend on AJAX, so if those work for you, this should too.

We will now attempt to detect how good your browser support is. If you get stuck, and want to get back to the old and busted frames site, you have to clear your cookies and start over at http://www.photoprism.net.
